# WideoWebPortfolio

Nowoczesna aplikacja webowa pełniąca rolę portfolio (z ukierunkowaniem na produkcję wideo oraz aplikacje internetowe). Projekt składa się z dwóch głównych modułów: nowoczesnego interfejsu (Single Page Application) oraz backendu (API i Panel CMS).

## Architektura i Technologie

*   **Frontend (SPA):** React.js, Vite, Tailwind CSS, Framer Motion.
*   **Backend (API & CMS):** ASP.NET Core MVC. Dostarcza dane dla interfejsu za pośrednictwem API (JSON) oraz służy jako zintegrowany System Zarządzania Treścią (CMS) dla administratorów.
*   **Baza Danych:** SQLite (dla łatwości uruchomienia deweloperskiego) obsługiwana przez Entity Framework Core.

## Struktura Katalogów

*   `Frontend/` - Kod źródłowy aplikacji React.
*   `Backend/` - Kod źródłowy API i Panelu CMS w środowisku ASP.NET Core.
